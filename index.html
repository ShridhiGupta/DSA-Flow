Linear Search

  • Time Complexity: O(n)
  • Space Complexity: O(1)
  • Best for: Small arrays or unsorted data
  • Process: Checks each element sequentially

Binary Search

  • Time Complexity: O(log n)
  • Space Complexity: O(1)
  • Requires: Sorted array
  • Process: Divides search interval in half repeatedly
